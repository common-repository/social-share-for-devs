A simple, lightweight plugin to add social share buttons to your website.

== Description ==

Add social share icons to your site without hassles. The Social Share plugin is designed as a very simple, lightweight and performant plugin for devs. Forget about taking care of a million settings. Just install it, select your favorite social networks and that's it!

Feature list

* Share the content of your website on the most important social networks (full list below)
* Customize buttons with predesigned styles or add your own classes to apply styles as you see fit.
* Options to specify sharing icon shape – Square, Round and Rectangular
* Place your buttons wherever you want using shortcodes.
* Compatible with Gutenberg blocks.

Supported Social Networks

This plugin is intended to be simple, very lightweight and easy to use. At this moment, we support the following social sharing sites and apps:

* Facebook
* Twitter
* LinkedIn
* WhatsApp
* Pinterest
* Email

== Installation ==

The Social Share for Devs plugin is very easy to use and you can get it up and running in a few minutes:

1. [Download](https://wordpress.org/plugins/social-share-for-devs "Download") the zip file
2. On your WordPress Dashboard, go to Plugins > Add new > Upload Plugin
3. Click on Choose File… and select the zip file you downloaded.
4. Activate the plugin

Alternatively, you can install the plugin directly from the WordPress Dashboard searching for Social Share for Devs Plugin by Wcanvas. 

Configuration

1. Select the social buttons you want to display on pages.
2. Select the type of button you want to display (Default, Rounded, Square Icon, Circle Icon, Custom)
3. Optional: add your class for CSS styles (for custom display)
4. Add the shortcode  [wc-ssb] to the page you want the buttons displayed on.

== Frequently Asked Questions ==

= Q. Can I add other social share sites? =

A. At the moment we only offer support for the sites listed above.
